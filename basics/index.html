Basic Usage

This documentation explains how to use orga to parse org-mode content into AST.Find the complete definition of AST here.

Install

npm install orga

Examples

Getting Started

Create a file test.js.
  const { parse } = require(`orga`)
  const content = "* TODO Hello World"
  const ast = parse(content)
The ast would look like this:
{ type: 'root',
  children:
   [ { type: 'section',
       children:
        [ { type: 'headline',
            children:
             [ { type: 'text',
                 children: [ [length]: 0 ],
                 value: 'Hello World',
                 parent: [Circular] },
               [length]: 1 ],
            level: 1,
            keyword: 'TODO',
            priority: undefined,
            tags: [ [length]: 0 ],
            parent: [Circular] },
          [length]: 1 ],
       level: 1,
       parent: [Circular] },
     [length]: 1 ],
  settings: {} }

Add Options

You can customize the parser by sending in options.
  const { parse } = require(`orga`)
  const content = "* NEXT Hello World"
  const ast = parse(content, { todos: ['TODO', 'NEXT', 'DONE'] })

User the Parser

  const { Parser } = require(`orga`)
  const content = "* NEXT Hello World"
  const parser = new Parser() // the Parser accept options too
  const ast = parser.parse(content)
